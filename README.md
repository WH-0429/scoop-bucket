# My Scoop Bucket

这是我的个人 Scoop 软件仓库。

## 已包含的软件

- **lceda-pro**: 嘉立创 EDA 专业版
- **Claude-code-desktop**: Claude code 桌面版
## 如何安装

1. 添加此仓库：

   ```powershell
   scoop bucket add WH-bucket https://github.com/WH-079/scoop-bucket
   ```
   嘉立创 EDA 专业版
   ```powershell
   scoop install WH-bucket/lceda-pro
   ```
   Claude code 桌面版
   ```powershell
   scoop install WH-bucket/Claude-code-desktop
   ```
   Codex 桌面版
   ```powershell
   scoop install WH-bucket/Codex-desktop
   ```
